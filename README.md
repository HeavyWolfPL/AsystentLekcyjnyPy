# AsystentLekcyjnyPy

Bot łączący się z API aplikacji Wulkanowy uzyskując w ten sposób plan lekcji.

## Instalacja
#### **Wymagania**
- Python 3.7.8+
- Discord.py 1.7.2+
- JSON Requests

#### **Tokeny**
Skopiuj plik token-example.json, wklej tam poprawne tokeny, a następnie zmień jego nazwę na `config.json`
