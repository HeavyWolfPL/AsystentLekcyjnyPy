# AsystentLekcyjnyPy

Bot łączący się z API aplikacji Wulkanowy uzyskując w ten sposób plan lekcji.

## Instalacja
#### **Wymagania**
- Python 3.7.8+
- Discord.py 1.7.3+
- Tabulate
- Vulcan-API

#### **Tokeny**
Skopiuj plik config-example.json, wklej tam poprawne tokeny, a następnie zmień jego nazwę na `config.json`
